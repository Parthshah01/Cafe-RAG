# Cafe RAG Assistant

A Retrieval-Augmented Generation (RAG) application built using:

- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Ollama

## Features

- PDF ingestion
- Vector search using FAISS
- Context retrieval
- AI-powered cafe assistant
